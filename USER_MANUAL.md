## 👤 顧客篇 (Customer Manual)

歡迎使用新泰汽車傳動軸預約系統。透過本平台，您可以輕鬆管理您的愛車維修計畫。

### 1. 登入系統
- **一般網頁版**：點擊導覽列右上角的 「Google 登入」按鈕。
- **LINE 官方帳號**：
    - 直接點擊 Rich Menu 選單，系統將自動為您登入。
    - 在 LINE 內開啟時，介面會自動隱藏多餘導航列，提供更流暢的操作體驗。
- 登入後即可開始預約或管理您的個人紀錄。

### 2. 線上預約流程
- **選擇服務**：在預約頁面中選擇您需要的維修項目。
- **選擇日期**：
    - 畫面顯示 **月曆視圖**，您可以透過箭頭切換月份。
    - 紅色標記表示店休，藍色標記表示補班，白色/灰色則為一般營業日。
- **選擇時段**：
    - 點擊可用日期後，系統會彈出 **時段選擇視窗**。
    - 系統會自動過濾已預設預約之時段與過去時間。
- **填寫資訊**：輸入車型、車牌號碼與聯絡電話。
- **確認預約**：提交後，預約狀態預設為「待確認」，管理員核准後會變更為「已確認」。

### 3. 管理我的預約
- 點擊頭像選單中的 **「我的預約」**。
- 您可以查看所有即將到來的維修時程。
- **取消預約**：若計畫有變，請在時段開始前點擊「取消預約」以釋出名額。

### 4. 保固查詢
- **訪客查詢**：無須登入，點擊導覽列 **「保固查詢」**，輸入「電話號碼」與「車牌號碼」即可查詢最新維修記錄。
- **會員查詢 (新功能)**：
    - 登入後，點擊 **「我的保固」** (或透過 LINE 選單進入)。
    - 系統會自動列出您名下所有「保固中」的維修卡，無須手動輸入資料。

## 🛠 管理者篇 (Admin Manual)

管理者擁有全站控制權，請妥善保管您的管理帳號。

### 1. 進入管理後台
- 以具備管理權限的 Google 帳號登入後，點擊導覽列右上角的 **「後台管理 (齒輪圖示)」**。

### 2. 工作看板 (Task Board)
- **日曆檢視**：使用 FullCalendar 查看每週/每日的維修負載。
- **清單檢視**：快速搜尋特定的車牌或電話。
- **狀態管理**：
    - **待確認 (Pending)**：新進預約，需點擊「核准預約」。
    - **已確認 (Confirmed)**：準備維修中。
    - **標記完工 (Completed)**：維修完成，點擊後需輸入「實際工時」以供後續分析。
    - **已取消 (Cancelled)**：若客戶取消，請務必手動標置，系統將自動釋出時段。

### 3. 假日管理 (Holiday Management)
- 進入「後台」-> **「假日管理」**。
- **自動同步**：點擊「同步最新假日」以自動抓取台灣行政機關辦公日曆。
- **手動調整 (加班/店休)**：
    - 點擊列表中的「營業/店休」按鈕，可直接切換該日期的開放狀態。
    - **手動新增**：若有臨時店休或活動，可點擊「手動新增」建立特定日期。
    - **刪除記錄**：點擊垃圾桶圖示可撤銷特定的假日設定，恢復為系統預設邏輯。

### 4. 系統設定
- **基本資料**：在「系統設定」中修改店名、電話、聯絡地址，全站資訊將同步更新。
- **排程參數**：可調整預設預約時段縮減或延長。

### 5. 維修項目管理
- 在「服務項目」中新增或修改維修名稱、建議價格與保固月份。
- 設定為「隱藏」的項目將不會出現在顧客預約選單中。
